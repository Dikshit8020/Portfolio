01 / Origin

How I Got Here

Started in 2nd year with an 8051 microcontroller — blinking LEDs, then reading ADC values from a temperature sensor. The moment I understood how timer interrupts work was when everything clicked. From there, I moved to STM32 — the jump from 8-bit to a Cortex-M4 at 168MHz with DMA, multi-timer, and hardware FPU was transformative.

My internship at Accurate Sensing Technologies cemented my skills. Writing calibration firmware for industrial thermal sensors — where a bug means a pyrometer reads 500°C when the actual temperature is 200°C — taught me the discipline of defensive programming, watchdog timers, and exhaustive boundary testing.

04 / Code Sample

Real Firmware I've Written

The FreeRTOS axis task from the robotic arm — each joint runs this template, receiving targets via a queue and signaling completion.

// axis_task.c — FreeRTOS motor controlembedded_c
/* One instance per axis (6 total).
   Receives target via xAxisQueue[id].
   Signals completion via EventGroup. */

void vAxisTask(void *pvParams) {
    AxisHandle_t *ax = (AxisHandle_t*)pvParams;
    int32_t target;

    for(;;) {
        xQueueReceive(xAxisQueue[ax->id],
                      &target, portMAX_DELAY);

        int32_t delta = target - ax->pos;
        HAL_GPIO_WritePin(ax->dir_port,
            ax->dir_pin,
            delta > 0 ? GPIO_PIN_SET : GPIO_PIN_RESET);

        for(uint32_t i = 0; i < abs(delta); i++) {
            HAL_GPIO_WritePin(ax->step_port,
                ax->step_pin, GPIO_PIN_SET);
            delay_us(ax->step_delay);
            HAL_GPIO_WritePin(ax->step_port,
                ax->step_pin, GPIO_PIN_RESET);
            delay_us(ax->step_delay);
        }

        ax->pos = target;
        xEventGroupSetBits(xDoneFlags,
                           (1 << ax->id));
    }
}
